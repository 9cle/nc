# N VS Code Extension

This extension adds basic language support for `.n` source files in Visual Studio Code.

## Features

- Syntax highlighting for N keywords, operators, strings, numbers, and comments
- `.n` file association
- Language configuration for comment and bracket pairing

## Installation

1. Open this workspace in VS Code.
2. Install this extension locally by using the `Developer: Install Extension from VSIX...` command after packaging, or add the `n-extension` folder to VS Code's extension development host.

## File support

- `.n` files are recognized as the N language
- `//` comments are highlighted
- strings support both single and double quotes
